Treasure Hunt

rules

  1. Type in a number between 5 and 50 to generate your gameboard (your number will be the number of generated fields)
  2. Roll the die - your pawn will move automatically by the number of steps you rolled
  3. If you step on a forest field, you can continue forward
  4. Watch out! If you step on the dark forest field, the game is over!
  5. To find the treasure, you have to step on the treasure field, which is always the last field of the board
  6. If you approach the end of the board and roll a number that exceeds the number of fields left, you will be transported back to field number one

Good luck!
